---

## Key Learnings

1. Spark performs significantly faster than MapReduce due to in-memory processing.
2. DataFrames are immutable and every transformation creates a new DataFrame.
3. Proper handling of null values improves data quality.
4. Aggregations help derive meaningful business insights.
5. Shuffle operations are expensive and should be minimized when possible.
