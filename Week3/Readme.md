# Task 3: Sales Data Analysis Using SQL

## Project Overview

This project analyzes the Superstore sales dataset using SQL concepts including Subqueries, Common Table Expressions (CTEs), Window Functions, JOINs, and Aggregations.

The objective is to transform raw sales data into meaningful business insights by creating normalized tables and performing analytical queries on customer sales performance.

---

## Objectives

* Import and store Superstore data in a database table.
* Create normalized tables from the raw dataset.
* Apply SQL Subqueries for advanced filtering.
* Use Common Table Expressions (CTEs) for aggregations.
* Implement Window Functions for ranking and analysis.
* Combine JOINs, CTEs, and Window Functions in a final analytical query.
* Generate customer sales insights to support business decision-making.

# Step 2: SQL Analysis Queries

## Query 1: Orders Above Average Sales

Technique Used:
Subquery

Purpose:
Identify orders whose sales value exceeds the average sales value across all orders.

Business Value:
Highlights high-performing transactions.

---

## Query 2: Highest Sales Order Per Customer

Technique Used:
Correlated Subquery

Purpose:
Find the largest order placed by each customer.

Business Value:
Helps understand customer purchasing behavior.

---

## Query 3: Total Sales Per Customer

Technique Used:
CTE

Purpose:
Calculate cumulative sales generated by each customer.

Business Value:
Measures customer contribution to revenue.

---

## Query 4: Customers With Above-Average Sales

Techniques Used:

* CTE
* Subquery

Purpose:
Identify customers whose total sales exceed the average customer sales.

Business Value:
Useful for loyalty programs and targeted marketing.

---

## Query 5: Customer Ranking Based on Total Sales

Technique Used:
Window Function (RANK)

Purpose:
Rank customers according to total revenue generated.

Business Value:
Helps identify top-performing customers.

---

## Query 6: Assign Row Numbers to Orders Within Customers

Techniques Used:

* ROW_NUMBER()
* PARTITION BY

Purpose:
Assign sequential numbers to orders for each customer.

Business Value:
Useful for customer order history analysis.

---

## Query 7: Top 3 Customers Based on Total Sales

Technique Used:
Window Function

Purpose:
Identify the three highest revenue-generating customers.

Business Value:
Supports VIP customer identification.
