# Basic Data Exploration and Cleaning using Pandas

## Objective
Learn Python basics and perform data exploration and cleaning using Pandas.



## Tasks Performed

1. Loaded CSV dataset into Pandas DataFrame
2. Explored dataset using:
   - head()
   - tail()
   - shape
   - columns
   - dtypes
3. Checked missing values
4. Filled/handled missing values
5. Filtered rows and selected columns
6. Removed duplicate rows
7. Created derived column:
   
   total_amount = Purchase Amount (USD) * quantity

8. Saved cleaned dataset as a new CSV file
